# 模板目录

用于存放按钮/元素截图模板，配合 CV 模板匹配使用。

## 命名规范

```
{element_name}.png
```

示例：
- `like_button.png` - 点赞按钮
- `comment_button.png` - 评论按钮
- `share_button.png` - 分享按钮
- `collect_button.png` - 收藏按钮
- `search_button.png` - 搜索按钮

## 制作方法

1. 截取按钮区域图像
2. 保存为 PNG 格式
3. 确保图像清晰、背景简单

## 使用方法

将模板图像放入此目录，SmartLocator 会自动加载。
